# 밍고(Mingo) 웹 배포 가이드 🚀

이제 밍고를 챗지피티(ChatGPT)처럼 누구나 키 입력 없이 쓸 수 있는 '진짜 웹 서비스'로 바꿀 시간입니다!

### 1단계: Vercel에 프로젝트 올리기
1.  [Vercel](https://vercel.com/)에 가입하고 로그인합니다.
2.  이 프로젝트 폴더(`siwoo`)를 자신의 GitHub에 업로드하거나, Vercel 대시보드에서 **"Add New" -> "Project"**를 누릅니다.
3.  밍고 프로젝트를 선택하고 **Deploy** 버튼을 누릅니다.

### 2단계: 핵심! 비밀 키(API Key) 설정하기
배포가 되면 처음에는 에러가 날 거예요. 시우님의 API 키가 서버에 등록되지 않았기 때문입니다.
1.  Vercel 대시보드에서 밍고 프로젝트의 **Settings** -> **Environment Variables**로 이동합니다.
2.  다음 내용을 입력하고 **Add**를 누릅니다:
    *   **Key**: `MINGO_MASTER_KEY`
    *   **Value**: (시우님의 구글 API 키를 여기에 넣으세요!)
3.  **Redeploy**를 하거나, 코드를 다시 한 번 push하면 끝!

---

### 🎉 이제 무엇이 달라지나요?
*   **키 입력 없음**: 이제 친구들이 접속해도 API 키를 물어보지 않습니다. 그냥 로그인만 하면 바로 밍고와 대화할 수 있어요!
*   **보안 철저**: 시우님의 소중한 API 키는 이제 서버 깊숙이 숨겨져 있어 아무도 훔쳐볼 수 없습니다.
*   **진짜 서비스**: `mingo-ai.vercel.app` 같은 멋진 주소로 친구들에게 공유해 보세요!

**문제가 생기면 언제든 말씀해 주세요! 밍고가 멋지게 날아오를 준비가 되었습니다!** 🐧🚀✨
